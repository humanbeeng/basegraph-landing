# SEO Optimizations for Basegraph Landing Page

## Overview
This document outlines all the SEO optimizations implemented for the Basegraph/Relay landing page.

## Completed Optimizations

### 1. **Comprehensive Metadata** ✅
- **Title Tags**: Implemented with template for dynamic pages
- **Meta Description**: Optimized with relevant keywords and clear value proposition
- **Keywords**: Added targeted keywords including:
  - AI code analysis
  - Linear integration
  - Production specs
  - Code context
  - Developer tools
  - AI agents
  - Codegraph
  - Static analysis
  - Software development
  - Ticket automation
  - Spec generation
  - Codebase mapping

### 2. **Open Graph Tags** ✅
- Full Open Graph implementation for social media sharing
- Configured for optimal display on Facebook, LinkedIn, etc.
- OG image placeholder ready at `/og-image.png` (1200x630px recommended)
- Proper locale and URL settings

### 3. **Twitter Card Tags** ✅
- Summary large image card type
- Twitter handle: @basegraph
- Optimized title and description for Twitter
- Twitter image configured

### 4. **Structured Data (JSON-LD)** ✅
- Schema.org SoftwareApplication markup
- Includes:
  - Application name and category
  - Description and features
  - Creator/Organization information
  - Social media links
  - Pricing information
- Helps search engines understand the product better
- Enables rich snippets in search results

### 5. **Sitemap** ✅
- Dynamic sitemap at `/sitemap.xml`
- Automatically generated by Next.js
- Includes proper change frequency and priority
- Points to: https://basegraph.app

### 6. **Robots.txt** ✅
- Located at `/public/robots.txt`
- Allows all crawlers
- Blocks API routes from indexing
- References sitemap location

### 7. **Semantic HTML** ✅
- Proper heading hierarchy (h1 → h2 → h3)
- ARIA labels for accessibility and SEO
- Section elements with proper labels
- Semantic structure throughout

### 8. **Technical SEO** ✅
- Canonical URL set to https://basegraph.app
- Language attribute set to "en"
- Mobile-friendly viewport configuration
- Fast loading with Next.js optimization
- Image optimization with Next.js Image component

### 9. **PWA Support** ✅
- Web manifest file (`/public/site.webmanifest`)
- Improves mobile SEO
- Enables "Add to Home Screen" functionality

### 10. **Performance & Core Web Vitals**
- Next.js automatic optimizations
- Image optimization with next/image
- Font optimization with next/font
- Vercel Analytics integrated
